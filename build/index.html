возможности

Оформление в торговой точке

Для компаний с кредитным отделом

Позволяет продавать в рассрочку/кредит при первом касании. Наиболее быстрый способ, требующий участия специалиста торговой организации.

Оформление черех call-центр

Для малого и среднего бизнеса

Позволяет продавать в рассрочку/кредит при очной работе с клиентом. Для подачи заявки не требуется наличие кредитного специалиста.

Mix технологий

Для удаленной работы с клиентом

Позволяет продавать в рассрочку/кредит, направляя клиенту ссылку или QR-код на подачу заявки. Наличие кредитного специалиста в организации при этом не требуется.

Оформление в e.com

Для онлайн продаж клиентам

Позволяет клиентам самостоятельно оформлять покупку, выбрав соответствующий способ оплаты на сайте:

  • в рассрочку/кредит
  • через баннер/ссылку/QR-код/корзину сайта
